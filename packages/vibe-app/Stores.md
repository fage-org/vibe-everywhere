## Full Description

Vibe is a secure mobile companion for AI coding workflows. It lets you monitor and control sessions
running on your own machine from iOS, Android, or web while keeping your code where it already
lives.

Important: Vibe is a companion app. It expects a compatible local runtime such as the Vibe CLI or
related agent tooling to be installed on your machine.

Key features:

- Continue coding sessions between desktop and mobile
- Receive push notifications for permission requests and task completion
- Access conversation history even when a terminal is offline
- End-to-end encryption for account linking and session data
- Real-time streaming of agent responses
- Open-source codebase with self-hosting-friendly architecture

How it works:
1. Install the Vibe runtime on your machine
2. Scan a QR code to securely pair devices
3. Start sessions from your machine and monitor them from Vibe
4. Switch between desktop and mobile control as needed

Vibe is built for developers who want remote visibility into coding agents without routing code
through a third-party relay. Your code stays on your machine; Vibe gives you the remote control.
