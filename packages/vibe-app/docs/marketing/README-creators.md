# Vibe Content Creator Brief

## Key Features

Vibe is an open-source mobile app that lets developers control AI coding agents from a phone, tablet,
or browser while the actual work runs on their own machine.

- Control Claude Code, Codex, and Gemini-compatible flows from iOS, Android, or web
- Runs on your hardware, not a relay SaaS
- End-to-end encrypted session and account linking
- Push notifications when agents need input or finish work
- Multiple concurrent sessions and machine switching
- Open-source codebase with self-hosting-friendly architecture

## Key Value Props

- I can monitor long-running coding work away from my desk
- My code stays on my machine while Vibe acts as the remote control surface
- I can answer permission prompts from my phone instead of returning to my laptop
- I can keep multiple agents moving in parallel without paying for a middleman relay
- I can audit and modify the full stack because it is open source

## Example Hooks

- "My coding agent needed sudo while I was getting coffee, so I approved it from my phone"
- "I keep my code on my own machine and still manage AI sessions anywhere"
- "Open-source mobile remote control for Claude Code, Codex, and more"
- "Why pay for a relay app when your own machine can do the work?"

## Requirements for Content

- Clearly say "Vibe" in the video or caption
- Mention that it is open source and runs on the user's own machine
- Show actual usage: pairing, notifications, session control, or remote monitoring
- Avoid implying that Vibe replaces code review or developer judgment

## Target Audiences

- Students and indie developers
- Remote and hybrid workers
- Parents and commuters
- Open-source enthusiasts who prefer owning their tools
