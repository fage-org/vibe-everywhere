# v0.1.19

- Add real-time chat detail refresh in the control app by consuming the relay SSE event stream for
  the currently open conversation instead of waiting for the 10-second polling cycle.
- Surface the most useful task failure reason in chat cards by preferring provider stderr and
  provider-originated error events over the generic terminal task error.
- Normalize common `gbt-*` model typos to `gpt-*` for Codex and OpenCode model profiles so invalid
  saved model ids no longer break chat requests.
